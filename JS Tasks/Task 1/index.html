The company needs to automate the customer registration process. Write a
function that will prompt the user for their last name, first name, and
email address. After the user's response, the function should return a
string of the format: "*Mail* - *LastName* *FirstName*". 

          
    function registerCustomer() {
      const lastName = prompt("Please enter your last name:");
      const firstName = prompt("Please enter your first name:");
      const email = prompt("Please enter your email address:");

      const registrationString = `${email} - ${lastName} ${firstName}`;       

      return registrationString;
    }
