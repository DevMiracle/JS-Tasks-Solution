There are several types of seats in the restaurant:
1. Small table - up to 2 people;
2. Medium table - up to 4 people;
3. Large table - up to 8 people.
The restaurant will not be able to accommodate a company of more than 8 people within one table :â€™-(
Write a function that will receive the number of guests from the user and suggest the most suitable table. 

          
    function suggestTable(numGuests) {
      if (numGuests <= 0) {
        return "Please enter a valid number of guests";
      } else if (numGuests <= 2) {
        return "You can be seated at a small table";
      } else if (numGuests <= 4) {
        return "You can be seated at a medium table";
      } else if (numGuests <= 8) {
        return "You can be seated at a large table";
      } else {
        return "Unfortunately, we cannot accommodate a group of your size within one table";      
      }
    }
