each registered customer. Once a month, a promotion is held, as a result some users
receive a discount. To become a winner, the sum of the digits of userâ€™s individual code

Write a function that will receive an individual code from the user and will return the sum of
the digits of this code. 

          
      function sumOfDigits(code) {
        let sum = 0;
        while (code) {
          sum += code % 10;
          code = Math.floor(code / 10);           
        }
        return sum;
